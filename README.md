# DecodeLabs-Project3
SQL-based analysis of sales data using MySQL, focusing on SELECT, WHERE, GROUP BY, ORDER BY, COUNT, SUM, and AVG operations.

This project focuses on SQL Data Analysis using MySQL. The objective was to extract meaningful business insights from a sales dataset by applying SQL queries and analytical techniques. Various operations such as data filtering, sorting, grouping, and aggregation were performed using SELECT, WHERE, ORDER BY, GROUP BY, COUNT, SUM, and AVG functions. The project demonstrates practical SQL skills for data exploration, reporting, and decision-making in real-world business scenarios.
